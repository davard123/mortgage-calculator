### 问题 2：主界面 5 个选项卡在手机上显示优化 ✅

**问题描述**：
- 5 个主选项（买卖房地产、重贷、收入计算、Closing Cost、About）在手机上分成多行显示
- 需要固定在界面底部，类似 App 底部导航栏

**修复内容**：

1. **CSS 样式优化**：
   ```css
   .bottom-nav-items {
       display: flex;
       justify-content: space-between;  /* 均匀分布 5 个按钮 */
       gap: 0;  /* 移除间距，确保一行显示 */
       max-width: 500px;
       margin: 0 auto;
   }
   
   .bottom-nav-btn {
       flex: 1;  /* 每个按钮平均分配空间 */
       min-width: 0;  /* 允许压缩 */
       padding: 6px 4px;  /* 紧凑间距 */
   }
   
   .bottom-nav-btn .nav-label {
       font-size: 10px;  /* 小字体适应手机 */
       white-space: nowrap;
       overflow: hidden;
       text-overflow: ellipsis;
   }
   ```

2. **手机端布局调整**：
   - 隐藏顶部标签栏（`.tab-nav { display: none !important; }`）
   - 显示底部固定导航栏
   - 添加 `body { padding-bottom: 70px; }` 防止内容被遮挡
   - 优化安全区域适配（`padding-bottom: max(6px, env(safe-area-inset-bottom, 0))`）

3. **视觉效果优化**：
   - 激活状态：图标和文字变为蓝色（而非白色背景）
   - 悬停效果：浅蓝色背景 + 圆角
   - 移除不必要的 transform 和 box-shadow，保持简洁
